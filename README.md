# Customer Behavior Analysis

## Project Overview
The objective of this project is to simulate a corporate-level, end-to-end data analytics workflow, demonstrating the ability to convert raw data into meaningful business insights.

- Data Preparation, Modeling & Exploratory Data Analysis (Python): Cleaned, transformed, and structured raw data to ensure it is analysis-ready and reliable.
- Data Analysis (SQL): Performed query-based analysis to simulate real-world business scenarios and extract insights related to customer behavior, segmentation, loyalty, and purchasing patterns.
- Visualization & Insights (Power BI): Developed an interactive dashboard to highlight key trends and performance metrics, enabling clear and effective data-driven decision-making.
- Reporting & Presentation: Compiled findings into a structured report and created a presentation to communicate insights and actionable recommendations to stakeholders in a clear and concise manner.

---

## Overview
The objective of this project is to transform raw customer transaction data into actionable insights by:

- Cleaning and preparing the dataset  
- Performing exploratory data analysis  
- Writing SQL queries to extract business insights  
- Building an interactive Power BI dashboard  
- Communicating findings through reports and visualizations

---

## Tools and Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL**
- **Power BI**
- **Jupyter Notebook**

---

## Workflow

### 1. Data Preparation & EDA (Python)
- Cleaned missing and inconsistent data  
- Converted data types  
- Performed exploratory data analysis to understand patterns and trends  

---

### 2. Data Analysis (SQL)
Performed business-focused analysis including:

- Revenue comparison by gender  
- Discount impact on purchasing behavior  
- Top-rated and most purchased products  
- Shipping type analysis  
- Customer segmentation (New, Returning, Loyal)  
- Subscription behavior analysis  
- Product performance within categories  

---

### 3. Data Visualization (Power BI)
- Built an interactive dashboard (`.pbix`)  
- Visualized:
  - Sales trends  
  - Customer segments  
  - Product performance  
  - Discount impact
 
  The interactive dashboard is included in this repository.

  File: `customer_behavior_dashboard.pbix`


---

### 4. Reporting & Insights
- Derived key insights from analysis  
- Identified high-value customers and top-performing products  
- Provided actionable business recommendations  

---

## Key Insights
- Customers using discounts often show higher purchase activity  
- Loyal customers contribute significantly to total revenue  
- Certain product categories dominate sales performance  
- Subscription status influences overall spending behavior  

---

## 🚀 How to Use
1. Open the dataset from `/data`  
2. Run the Jupyter Notebook for data cleaning and EDA  
3. Execute SQL queries for analysis  
4. Open the Power BI file to explore the dashboard
