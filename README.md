# Customer Behavior Analysis

## Project Overview
The objective of this project is to simulate a corporate-level, end-to-end data analytics workflow, demonstrating the ability to convert raw data into meaningful business insights.

- Data Preparation, Modeling & Exploratory Data Analysis (Python): Cleaned, transformed, and structured raw data to ensure it is analysis-ready and reliable.
- Data Analysis (SQL): Performed query-based analysis to simulate real-world business scenarios and extract insights related to customer behavior, segmentation, loyalty, and purchasing patterns.
- Visualization & Insights (Power BI): Developed an interactive dashboard to highlight key trends and performance metrics, enabling clear and effective data-driven decision-making.
- Reporting & Presentation: Compiled findings into a structured report and created a presentation to communicate insights and actionable recommendations to stakeholders in a clear and concise manner.

## Overview
